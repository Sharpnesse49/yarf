## Yarf (Yet Another Rust Fetcher)
*Another rust fetcher made to be fast &amp; light (~0m0,003s)*

<img width="560" height="725" alt="Capture d’écran du 2026-07-19 07-24-48" src="https://github.com/user-attachments/assets/6d23eef9-5e60-46ff-bb6d-a13cab04f870" />



Basic Benchmark :
real	0m0,003s
user	0m0,001s
sys	0m0,001s

### How To Install :
`git clone https://github.com/Sharpnesse49/yarf && cd yarf && cargo build && cd`
### Usage :
`./yarf/target/debug/yet-another-rust-fetcher`
### How To Change The Color :
*At the end of main.rs you can see all the available color, just uncomment the color you want AND comment the other line.*
